# Object_Battle_RPG
 
RPGチックな簡単なアプリゲームです。
ゲームを開始するとマップ上に主人公が出現します
一定の歩数を歩くとモンスターとエンカウントします。
エンカウント後はバトルシーンに移行します。

# Note
 
最後にエンカウントするドラゴンは通常のプレイでは絶対に倒せない設計にしています。
ゲームを解析して倒す手法を確立してみましょう。

# Usage
 
リポジトリ内に存在する「Object_Battle_RPG.apk」をAndroid端末にInstallしてゲームを開始してください。
